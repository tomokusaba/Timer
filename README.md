# 🗂️ Defrag Timer（デフラグタイマー）

Windows 95/98 時代の懐かしいディスクデフラグツールの画面を再現したタイマーアプリケーションです。

![.NET](https://img.shields.io/badge/.NET-10.0-512BD4?logo=dotnet)
![WPF](https://img.shields.io/badge/WPF-Windows-0078D4?logo=windows)
![License](https://img.shields.io/badge/license-MIT-green)

## 📸 スクリーンショット

アプリを起動すると、レトロなデフラグ画面が表示されます。タイマーの進行に合わせて、断片化したブロックが最適化されていく様子をアニメーションで楽しめます。

## ✨ 特徴

- 🎨 **レトロなUI**: Windows 95/98 のディスクデフラグメンターを忠実に再現
- ⏱️ **カスタマイズ可能なタイマー**: 分・秒単位で自由に時間を設定可能
- 🔄 **リアルなアニメーション**: 
  - 🔴 **赤**: システムファイル（移動不可）
  - 🔵 **青**: 断片化ファイル
  - 🟦 **シアン**: 連続データ（未処理）
  - 🟢 **緑**: 最適化済み
  - ⬜ **白**: 空き領域
  - 🟡 **黄色**: 移動中
  - 🟠 **オレンジ**: 読み取り中
- ⏸️ **一時停止・再開機能**: いつでも一時停止してから再開可能
- 🔁 **リセット機能**: 新しい断片化パターンで再スタート

## 🚀 必要要件

- [.NET 10.0 SDK](https://dotnet.microsoft.com/download/dotnet/10.0) 以降
- Windows OS（WPF アプリケーション）

## 📦 インストールと実行

```bash
# リポジトリをクローン
git clone https://github.com/yourusername/Timer.git
cd Timer

# ビルドして実行
dotnet run --project Timer
```

## 🎮 使い方

1. **時間設定**: 右下の「分」と「秒」のテキストボックスに希望の時間を入力
2. **開始**: 「開始(S)」ボタンをクリックしてタイマーを開始
3. **一時停止**: 実行中に「一時停止」ボタンをクリックで一時停止
4. **リセット**: 「リセット(R)」ボタンで新しいパターンで再スタート

## 🎯 アルゴリズム

デフラグアニメーションは以下のロジックで動作します：

1. **初期化**: ランダムに断片化パターンを生成（システム領域、断片化ファイル、連続データ、空き領域）
2. **進行**: タイマーの進捗に応じて、前方のデータブロックから順に「最適化済み（緑）」に変換
3. **空き領域の移動**: 進捗に合わせて空き領域が後方に集約される
4. **アニメーション**: 読み取り→移動→書き込みのサイクルを表現
